PiPara Shared Library and CLI
=============================

## Build
  1. Clean this directory. (as needed)
> make clean
  1. Build.
> make

## Install
  1. Install files as root.
> sudo make install

## Usage
  1. For example, output 0x7f.
> sudo piparactl 7f

## Shared Library (libpipara.so)
  * See "piparactl.c".
